## 4. Kế hoạch thực hiện trong 3 tuần

### Tuần 1: Khảo sát và phân tích yêu cầu

Công việc:

- Khảo sát quy trình nghỉ phép trong doanh nghiệp
- Phân tích yêu cầu hệ thống
- Xác định actor và use case
- Vẽ UML diagram

AI hỗ trợ:

- Gợi ý use case
- Hỗ trợ nghiên cứu tài liệu
- Hỗ trợ cấu trúc tài liệu

Người thực hiện:

- Phân tích yêu cầu
- Chọn lọc nội dung
- Vẽ sơ đồ UML

Kết quả:

- Danh sách yêu cầu
- Use case
- UML Diagram

---

### Tuần 2: Thiết kế hệ thống và lập trình

Công việc:

- Thiết kế Database
- Thiết kế UI
- Thiết kế API
- Xây dựng chức năng chính

AI hỗ trợ:

- Gợi ý thiết kế
- Hỗ trợ sinh code

Người thực hiện:

- Chỉnh sửa thiết kế
- Tích hợp hệ thống

Kết quả:

- Database
- Giao diện
- Chức năng chính

---

### Tuần 3: Kiểm thử và hoàn thiện

Công việc:

- Kiểm thử

- Hoàn thiện sản phẩm
- Release

- Gợi ý sửa lỗi
- Hỗ trợ cải tiến

Người thực hiện:

- Kiểm thử thực tế
- Đánh giá kết quả

Kết quả:

- Hệ thống hoàn thiện
- Demo sản phẩm
