# 2. Phân công vai trò trong quá trình thực hiện dự án

## 2.1. Vai trò của sinh viên

* Khảo sát và phân tích yêu cầu của hệ thống.
* Xác định các tác nhân và chức năng chính của hệ thống.
* Thiết kế sơ đồ Use Case Diagram, Sequence Diagram và Class Diagram.
* Thiết kế giao diện người dùng phù hợp với nghiệp vụ thực tế.
* Xây dựng giao diện bằng HTML, CSS và JavaScript.
* Kiểm thử hệ thống và xử lý các lỗi phát sinh.
* Hoàn thiện báo cáo thực tập và quản lý repository trên GitHub.

## 2.2. Vai trò của AI hỗ trợ

* Hỗ trợ xây dựng kế hoạch thực hiện dự án.
* Gợi ý cấu trúc tài liệu phân tích hệ thống.
* Hỗ trợ xây dựng và hoàn thiện các sơ đồ UML.
* Hỗ trợ giải thích lỗi trong quá trình lập trình.
* Hỗ trợ rà soát và hoàn thiện báo cáo thực tập.
* Hỗ trợ tổ chức và hoàn thiện GitHub Repository.
