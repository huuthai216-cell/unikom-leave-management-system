# Unikom Leave Management System

Hệ thống quản lý đề xuất và phê duyệt nghỉ phép trong công ty Unikom

## Giới thiệu

Hệ thống hỗ trợ nhân viên tạo yêu cầu nghỉ phép và hỗ trợ quản lý thực hiện phê duyệt, theo dõi trạng thái nghỉ phép.

## Chức năng chính

- Đăng nhập hệ thống
- Tạo đề xuất nghỉ phép
- Xem trạng thái đề xuất
- Phê duyệt nghỉ phép
- Từ chối đề xuất
- Quản lý nhân viên
- Xem báo cáo

## Công nghệ sử dụng

- GitHub
- UML
- Database
- API
